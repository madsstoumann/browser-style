  /snippets/performance

  - checkPageSize - Curl page, measure HTML/CSS/JS sizes, total payload
  - checkCompression - Test gzip/brotli headers via curl
  - checkCaching - Analyze cache-control, expires, etag headers
  - checkRedirects - Follow redirect chains, measure redirect count/time
  - checkImages - Extract image URLs, check formats (WebP support), sizes
  - checkFonts - Detect web font usage, loading strategy, formats

  /snippets/infrastructure

  - checkSSL - Use openssl/curl to validate certificates, expiry dates
  - checkDNS - Use dig/nslookup for DNS records, MX, SPF, DKIM validation
  - checkUptime - Simple curl-based availability testing with response codes
  - checkPorts - Use netcat/telnet to test open ports (80, 443, etc.)
  - checkCDN - Detect CDN usage from headers, test geographic endpoints

  /snippets/content

  - checkLinks - Extract all links with grep/sed, test each with curl
  - checkImages - Find img tags, test image URLs, check alt attributes
  - checkMeta - Extract meta tags (title, description, keywords)
  - checkSitemap - Download and validate XML sitemap structure
  - checkRobots - Fetch and analyze robots.txt files
  - checkLanguage - Detect lang attributes, hreflang tags

  /snippets/mobile

  - checkViewport - Extract and validate viewport meta tags
  - checkManifest - Download and validate PWA manifest.json files
  - checkAppleTouch - Check apple-touch-icon links and files
  - checkAMP - Detect AMP implementation, validate AMP tags

  /snippets/technical

  - checkHTML - Basic HTML structure validation with grep/regex
  - checkCSS - Extract CSS links, test loading, basic validation
  - checkJS - Find script tags, test JavaScript file availability
  - checkFavicon - Test favicon.ico, PNG variants, sizes
  - checkCanonical - Extract and validate canonical tags
  - checkHreflang - International SEO hreflang validation

  /snippets/monitoring

  - checkErrors - Look for common error patterns in HTML output
  - checkConsole - Scan for JavaScript console.log/error patterns
  - checkDeprecated - Find deprecated HTML tags/attributes
  - checkMixed - Detect mixed content (HTTP resources on HTTPS pages)

  /snippets/compliance

  - checkCookies - Extract cookie usage, detect tracking scripts
  - checkPrivacy - Scan for privacy policy links, GDPR banners
  - checkGoogleServices - Detect Google Analytics, Fonts, Tag Manager
  - checkSocialMedia - Find social media integration, sharing buttons

  These scripts would use standard bash tools:
  - curl - Fetch content, check headers, test URLs
  - grep/sed/awk - Parse HTML, extract patterns
  - jq - Parse JSON responses (APIs, manifests)
  - openssl - SSL certificate analysis
  - dig/nslookup - DNS queries
  - head/tail - File analysis
  - Regular expressions for pattern matching
