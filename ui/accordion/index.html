Accordion — CSS-only

An accordion is a collapsible content panel that allows users to toggle the visibility of sections of content. This accordion is built with a details element containing a summary and a div.
It is not using JavaScript. Use the name-attribute to autocollapse in supported browsers.

Composable Variants

Variants can be combined via the space-separated variant attribute.

variant="bordered"

First item
Border around the entire group — inline borders on items, block borders on first and last.
Second item
Each item shares the outer border frame.
Third item
The last item closes the frame with a bottom border.

variant="bordered divided"

First item
Bordered group with divider lines between items.
Second item
Each item is separated by a horizontal line.
Third item
Divided adds a border-block-end on each item.

variant="bordered rounded"

First item
Bordered group with rounded corners on first and last items.
Second item
Middle items have no border radius.
Third item
Last item gets bottom border radius.

variant="separated rounded"

First item
Card-style with gap between items and rounded corners on each card.
Second item
Each item is a standalone card with full border and larger radius.
Third item
Uses --ui-accordion-border-radius-separated for a larger radius.

variant="breakout bordered rounded"

First item
Open items shift out via translate. The open item gets a thicker border and adjacent items react.
Second item
Click to see the breakout effect — this item shifts down when the previous one opens.
Third item
Items further from the open item shift more.
Fourth item
The breakout variant works best with bordered and rounded.

variant="breakout elevated rounded"

First item
Elevated uses box-shadow instead of borders on the open item. Requires breakout.
Second item
Adjacent items get a lighter shadow to create a depth hierarchy.
Third item
This combines well with rounded for a modern card-like appearance.

variant="background bordered divided rounded"

First item
The background variant adds a container background color and padding.
Second item
Combined with bordered, divided, and rounded.
Third item
The background uses --ui-accordion-background.
