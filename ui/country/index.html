UI: Components

Country

Country is a collection of flags from around the world. Each flag is an SVG image with a unique filename that corresponds to the country's ISO 3166-1 alpha-3 code.

Comes with varous options: circle, rounded, square, wavy.

Note, that the options might not work with all flags, as some flags are not designed to be displayed in a circle or a square.
