Animation Multipliers

Use a multiplier (1–3) to control animation intensity. Default is 1×.

fade-up — Default vs 2× vs 3×

2× intensity (220px translation). <lay-out animate="fade-up(2)" lg="columns(3)">
3× intensity (330px translation). <lay-out animate="fade-up(3)" lg="columns(3)">

Custom Easings

Apply custom easings to item animations with the easing attribute.

slide-in() with easing="ease-elastic-1"

Items slide in with elastic overshoot. <lay-out animate="slide-in()" easing="ease-elastic-1" lg="columns(3)">

fade-up() with easing="ease-spring-1"

Items spring up gently. <lay-out animate="fade-up()" easing="ease-spring-1" lg="columns(3)">

Pace — Exit

Add an exit token to reverse item animations on scroll-out. Exit stagger is reversed — last child exits first.

fade-up() pace="exit"

Items fade up on entry, reverse on exit. <lay-out animate="fade-up()" pace="exit" lg="columns(3)">

zoom-in() pace="exit"

Items zoom in on entry, reverse on exit. <lay-out animate="zoom-in()" pace="exit" lg="columns(3)">

flip-up() pace="exit-fast"

Items flip up on entry, snappy reverse on exit. <lay-out animate="flip-up()" pace="exit-fast" lg="columns(3)">

reveal-circle() pace="exit-slow"

Items reveal on entry, slow reverse starting in contain phase. <lay-out animate="reveal-circle()" pace="exit-slow" lg="columns(3)">

zoom-in() pace="exit"

Items zoom in on entry, reverse on exit. <lay-out animate="zoom-in()" pace="exit" lg="columns(3)">
