When a card has exactly one link and the clickable attribute, the entire card becomes a click-through. Use clickable="hide" to visually hide the link text.

Split Ratios

Horizontal layouts support custom column ratios via the split() token.

split(1/1) — 50/50
split(1/2) — 33/66
split(2/1) — 66/33

Subgrid Alignment

Using CSS subgrid, content fields align vertically across cards — even when headlines have different lengths. Cards must have a consistent number of content elements (eyebrow, headline, summary, tags, link = 5).

Responsive — Container Queries

Drag the resize handle to see the card adapt. The same card changes layout based on its own width, not the viewport.

card="vertical ar(1/1)" card-md="horizontal ar(4/3)" card-xl="overlay(bl) ar(21/9)"
card="vertical" card-md="horizontal split(1/1)" card-xl="horizontal-r split(2/1)"
