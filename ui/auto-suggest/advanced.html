UI: AutoSuggest - Advanced Examples

← Back to basic examples

These examples demonstrate all search modes using a local products.json file with 20 sample products.

1. Local Mode - Simple Substring Search

Loads data once, searches with simple includes() matching.

Try: "apple", "headphones", "camera"

2. Local Keywords Mode - Multi-Column Search (AND)

Searches across multiple fields: name, brand, category, and description.

All terms must match (AND logic): "apple laptop" finds MacBook Pro

Try: "apple laptop", "sony headphones", "wireless charging"

3. Local Keywords Mode - Multi-Column Search (OR)

Searches across name, brand, and category.

Any term can match (OR logic): "apple sony" finds all Apple OR Sony products

Try: "apple sony", "laptop phone", "nike lululemon"

4. Local Fuzzy Mode - Typo Tolerant (Strict)

Fuzzy matching with high threshold (min-score: 0.7) - requires close similarity

Try typos: "mackbook" → MacBook, "hedphones" → headphones, "camra" → camera

5. Local Fuzzy Mode - Typo Tolerant (Lenient)

Fuzzy matching with low threshold (min-score: 0.5) - very forgiving

Try worse typos: "mcbok" → MacBook, "hdfone" → headphones, "ofice chair" → office chair

6. Local Fuzzy Mode - Multi-Word with Typos

Fuzzy matching on multiple words with typos tolerated

Try: "wireles charger", "gamin mouse", "espreso machine"

7. Category-Based Keyword Search

Hierarchical category search: "electronics computers laptops" finds all laptops

Try: "home kitchen coffee", "sports outdoors fitness", "electronics audio headphones"

8. Manual Mount Example with Shadow DOM

This example demonstrates:

  • Manual mounting with nomount attribute
  • Shadow DOM rendering (no noshadow)
  • Custom styles with styles attribute
  • Form integration with name attribute

After selecting, access in console: document.getElementById('manual-form').elements.product.value

Data Structure

All examples use products.json with this structure:

{
  "id": "1",
  "name": "MacBook Pro",
  "category": "Electronics > Computers > Laptops",
  "brand": "Apple",
  "description": "High-performance laptop for professionals",
  "price": 2499,
  "sku": "MBP-16-2024"
}
