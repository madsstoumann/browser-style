# Changelog

All notable changes to the Analog Clock Component component will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0] - 2025-12-01

### Changed
- Added calude.md to analog-clock and data-chart, fixed a minor issue in dev-to

## [1.1] - 2025-03-01

### Changed
- Add digital clock component and refactor related files for improved structure and styling
- Refactor analog clock component: streamline date formatting and timezone handling; update package version

## [1.2] - 2025-02-01

### Changed
- Add new UI analog clock component with CSS styling and JavaScript initialization
- Add support for opposite color-scheme and steps attribute in analog clock component
- Add analog clock component with customizable features and styles
- Enhance analog clock component with customizable markers and improved styling options
- Refactor analog clock component styles for improved marker handling and layout consistency
- Remove deprecated analog clock HTML file and update analog-clock component usage with date attribute
- Refactor analog clock styles and clean up async loader CSS for improved readability and maintainability
- Update package versions and add new module links for analog clock and async loader components
- Update package versions and add dev-to module in package-lock.json
