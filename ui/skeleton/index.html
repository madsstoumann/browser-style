Skeleton

Skeleton is a technique used to provide a placeholder for content that is still loading. It is a visual representation of the layout of a page, showing where elements will be placed and how they will be styled. Skeletons are often used to improve the user experience by reducing the perceived loading time of a page. Skeletons are typically made up of simple shapes and lines that mimic the layout of the final page.

Build a skeleton using grid and aspect-ratio and border-radius utility-classes.
Used here with midifiers --wave and --pulse
